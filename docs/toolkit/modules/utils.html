GPU Memory Management

Force GPU Memory Release

This script attempts to forcibly free GPU memory by terminating processes that might be holding onto GPU resources. Useful for recovering from CUDA out-of-memory errors or orphaned processes that don’t release GPU memory.

histtext_toolkit.utils.clear_gpu_memory.list_gpu_processes()[source]

List all processes using the GPU.

Uses nvidia-smi to identify processes utilizing GPU resources and displays detailed information about each process.

Returns:

List of process IDs (PIDs) currently using the GPU

Return type:

List[int]

histtext_toolkit.utils.clear_gpu_memory.kill_process(pid, force=False)[source]

Kill a process by its PID.

Terminates a process using either SIGTERM (graceful) or SIGKILL (force), depending on the specified option.

Parameters:

  • pid (int) – Process ID to terminate

  • force (bool) – Whether to force kill with SIGKILL (9) instead of SIGTERM (15)

Returns:

True if the process was successfully terminated, False otherwise

Return type:

bool

histtext_toolkit.utils.clear_gpu_memory.restart_display_manager()[source]

Restart the display manager to release GPU resources.

Attempts to identify and restart the active display manager. This will restart the X server, which will free GPU resources, but will also close all GUI applications. Use with caution.

Return type:

None

histtext_toolkit.utils.clear_gpu_memory.check_gpu_memory()[source]

Check current GPU memory usage.

Queries the GPU for its current memory usage statistics using nvidia-smi.

Returns:

(used_memory, total_memory) in MB

Return type:

Tuple[int, int]

histtext_toolkit.utils.clear_gpu_memory.free_gpu_memory(force=False, restart_x=False)[source]

Try to free GPU memory by terminating processes.

Main function that orchestrates the process of identifying and terminating GPU-using processes. First targets Python processes, then offers to terminate other processes if memory is still in use.

Parameters:

  • force (bool) – Whether to use SIGKILL instead of SIGTERM for immediate termination

  • restart_x (bool) – Whether to offer restarting the X server if memory is still high

Return type:

None
